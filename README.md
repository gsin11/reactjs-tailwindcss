# ReactJS TailwindCSS component library

## Installation
`npm i reactjs-tailwindcss`

## How to use
`import { Toggle } from "reactjs-tailwindcss"`

```jsx
<Toggle 
  ariaLabel="toggle theme" 
  onChange={()=>{}} 
  isChecked={false} 
  className="green"
/>
```
